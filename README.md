## 下载

### Docker

```
docker pull apachecn0/flygon-golang-zh
docker run -tid -p <port>:80 apachecn0/flygon-golang-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-golang-zh
flygon-golang-zh <port>
# 访问 http://localhost:{port} 查看文档
```
